# 📊 Análise de Vendas com Python

## 📌 Sobre o Projeto
Projeto de análise exploratória de dados utilizando Python, com foco em métricas de vendas e geração de insights para apoio à tomada de decisão.

## 🛠 Tecnologias Utilizadas
- Python
- Pandas
- Matplotlib

## 📊 Análises Realizadas
- Cálculo do faturamento total
- Identificação do produto mais vendido
- Faturamento por categoria
- Faturamento por região
- Cálculo do ticket médio
- Visualização gráfica do faturamento por categoria

## 🎯 Principais Insights
- A categoria **Eletrônicos** apresentou o maior faturamento.
- A região **Sudeste** concentrou o maior volume financeiro.
- O ticket médio indica predominância de produtos de maior valor agregado.

## 📁 Estrutura do Projeto
- `analise_vendas.py` → Script contendo criação dos dados, tratamento, análise e visualização.
