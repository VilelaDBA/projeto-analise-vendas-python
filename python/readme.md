# Análise de Vendas com Python

## Sobre o Projeto
Análise exploratória de dados de vendas utilizando Python, com cálculo de faturamento, ticket médio, produtos mais vendidos e visualizações gráficas.

## Tecnologias
- Python
- Pandas
- Matplotlib

## Como Rodar
1. Instale as dependências: `pip install pandas matplotlib`
2. Execute: `python analise_vendas.py`
